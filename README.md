# ecommerce-analysis-portfolio

## Price vs Purchase Behavior Analysis

This project analyzes how purchase behavior differs by item category
based on price level, transaction share, and repurchase rate.

### Key Focus
- Price level differences by item category
- Transaction share as a measure of purchase frequency
- Repurchase rate as a supporting metric to interpret purchase behavior

### Tools
- Python
- pandas
- matplotlib

This analysis was conducted as part of a data analysis portfolio project.
